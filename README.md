# Rubini Raju - Portfolio

Welcome to my portfolio website! This is a modern, professional portfolio built with HTML, CSS, and JavaScript.

## Features

- 📱 Fully Responsive Design
- 🎨 Modern & Professional UI
- ⚡ Smooth Scrolling Navigation
- 📧 Contact Form
- 🖼️ Project Showcase
- 💼 Skills Section
- 📄 Resume Download
- ✨ Animations & Transitions

## Sections

1. **About** - Brief introduction about yourself
2. **Projects** - Showcase your work and projects
3. **Skills** - Display your technical skills
4. **Contact** - Get in touch form
5. **Resume** - Link to download your resume

## Getting Started

1. Clone this repository
2. Open `index.html` in your browser
3. Customize the content with your own information
4. Add your projects, skills, and experience

## Customization

- Edit `index.html` to update content
- Modify `styles.css` to change colors and styling
- Update `scripts.js` for additional functionality

## Colors Used

- Primary: #2c3e50 (Dark Blue-Gray)
- Secondary: #3498db (Blue)
- Accent: #e74c3c (Red)

## Deployment

Deploy to GitHub Pages:
1. Push your code to GitHub
2. Go to Settings → Pages
3. Select main branch as source
4. Your site will be live!
